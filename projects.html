1. Wireless Communication

1.1 RF Energy Harvesting System

Designed and fabricated an RF energy harvesting system using a patch antenna and rectifier circuit to harvest ambient Wi-Fi signals and power an LED without external power supply.

1.2 Optimization of Microstrip Patch Antenna using MATLAB

Optimized microstrip patch antenna parameters using MATLAB simulations to improve return loss, bandwidth, and VSWR.

1.3 Optimization of Microwave Filter

Designed and optimized microwave filter structures using MATLAB/CST tools to enhance frequency selectivity and insertion loss.

1.4 Patch Antenna Calculator Website

Developed a responsive web-based calculator using HTML, CSS, and JavaScript to compute patch antenna dimensions based on operating frequency and substrate parameters.

2. Embedded Systems

2.1 Heart Beat Counter using Pulse Sensor Module

Designed a heart beat monitoring system using a pulse sensor module and microcontroller to measure and display heart rate.

2.2 Li-Fi Based Data Transmission System

Implemented Li-Fi based data transmission using LED as transmitter and photodiode as receiver, enabling wireless communication using visible light.

2.3 Weather and Location Monitoring System

Built an embedded system to monitor environmental parameters such as temperature and humidity along with location tracking using sensor and GPS modules.

2.4 Development Board using 8051 Microcontroller

Designed and implemented a custom controller board based on the 8051 microcontroller for learning and control applications.

3. VLSI Projects (Cadence Virtuoso)

3.1 Design and Analysis of 6T SRAM Cell

Designed and simulated a 6T SRAM cell using Cadence Virtuoso and analyzed read stability, write margin, and noise margins.

3.2 Radiation Hardened Circuit using DICE Architecture

Designed a radiation-hardened circuit using Dual Interlocked Storage Cell (DICE) architecture to mitigate single event upsets in space and high-radiation environments.
