1. Embedded Systems

1.1 Weather and Location Monitoring System

This project focuses on the design and implementation of an embedded monitoring system capable of measuring environmental parameters and geographical location in real time. Sensors were interfaced with a microcontroller to continuously monitor temperature and humidity, while a GPS module was used to acquire precise location coordinates.

The system was designed with reliability and real-world deployment in mind, ensuring stable sensor readings, efficient data acquisition, and proper handling of serial communication with the GPS module. This project strengthened my understanding of sensor interfacing, serial protocols, embedded firmware design, and real-time data processing, making it suitable for applications such as environmental monitoring, logistics tracking, and IoT-based field systems.

1.3 Heart Beat Counter using Pulse Sensor Module

This project involves the design of a heart rate monitoring system using a pulse sensor module interfaced with 8051 microcontroller. The sensor detects variations in blood flow and converts them into electrical signals, which are processed to calculate the heart rate in beats per minute (BPM).

The system was programmed to filter noise, count pulses accurately, and display the measured heart rate in real time. Through this project, I gained practical experience in analog signal handling, sensor calibration, embedded data processing, and basic biomedical instrumentation concepts, making it relevant for healthcare monitoring and wearable device applications.

1.4 Li-Fi Based Data Transmission System

This project demonstrates a Li-Fi (Light Fidelity) based wireless communication system that uses visible light for data transmission. An LED was used as the transmitter to modulate digital data, while a photodiode at the receiver end detected the variations in light intensity and converted them back into electrical signals.

The system highlights the feasibility of using visible light as an alternative to traditional RF communication, especially in environments where RF interference is undesirable. This project provided valuable insights into optical wireless communication, modulation techniques, signal detection, and practical challenges such as ambient light interference and line-of-sight constraints.

2. VLSI Projects (Cadence Virtuoso)

2.1 Design and Analysis of 6T SRAM Cell

This project focuses on the design and simulation of a 6-transistor (6T) SRAM cell using Cadence Virtuoso. The SRAM cell was carefully designed to achieve stable read and write operations while maintaining low power consumption and reliable data retention.

Detailed analysis was performed to evaluate key performance parameters such as read stability, write margin, and static noise margin (SNM). Through extensive simulations, I developed a deeper understanding of CMOS memory design, device sizing trade-offs, and the impact of transistor parameters on memory performance, which are critical aspects of modern VLSI design.

3. Wireless Communication

3.1 RF Energy Harvesting System

This project focuses on harvesting ambient RF energy from Wi-Fi signals using a patch antenna and rectifier circuit. The captured RF signals were converted into DC power, which was used to drive a low-power LED without any external power source.

The system design involved antenna selection, impedance matching, and rectifier circuit optimization to maximize power conversion efficiency. This project provided practical exposure to RF circuit design, energy harvesting concepts, and low-power electronics, highlighting the potential of self-powered wireless devices for future IoT applications.

3.2 Optimization of Microstrip Patch Antenna using MATLAB

In this project, I optimized the design parameters of a microstrip patch antenna using MATLAB simulations. The objective was to improve key antenna performance metrics such as return loss, bandwidth, and Voltage Standing Wave Ratio (VSWR) for a target operating frequency.

By systematically varying antenna dimensions and substrate properties, I analyzed their impact on radiation characteristics and impedance matching. This project strengthened my understanding of antenna theory, electromagnetic behavior, and numerical analysis, while also demonstrating the use of MATLAB as a powerful tool for RF design optimization.

3.3 Optimization of Microwave Filter

This project involves the design and optimization of microwave filter structures to achieve improved frequency selectivity and reduced insertion loss. MATLAB and electromagnetic simulation tools were used to analyze filter responses and refine design parameters.

The project emphasized the importance of filter design in RF and microwave communication systems, particularly for interference rejection and signal conditioning. Through this work, I gained hands-on experience in microwave theory, frequency-domain analysis, and performance optimization of RF components.

This project is a web-based patch antenna calculator developed using HTML, CSS, and JavaScript to assist engineers and students in quickly estimating antenna dimensions. The tool computes parameters such as patch width and length based on operating frequency and substrate properties.

The website was designed to be responsive, user-friendly, and accurate, combining web development skills with RF engineering concepts. This project reflects my ability to bridge core electronics knowledge with software development to create practical engineering tools.
